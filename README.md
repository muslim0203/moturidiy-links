# Imom Moturidiy xalqaro ilmiy-tadqiqot markazi – link in bio

O‘zbekiston Respublikasi Vazirlar Mahkamasi huzuridagi **Imom Moturidiy xalqaro ilmiy-tadqiqot markazi**ning rasmiy “link in bio” sahifasi.

Yagona `index.html` fayl – tashqi kutubxonalarsiz, mobil uchun moslashtirilgan, islomiy me’moriy bezak ruhida yaratilgan.

## Texnologiyalar
- Sof HTML5, CSS3, JavaScript
- Glassmorphism, SVG geometrik naqshlar, CSS animatsiyalar
- Hech qanday framework ishlatilmagan

## Mahalliy ishga tushirish
Shunchaki `index.html` faylni brauzerda oching.

## Deploy
GitHubga push qilingach, [vercel.com/new](https://vercel.com/new) orqali reponi tanlab bir bosish bilan statik sayt sifatida joylash mumkin.
